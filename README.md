>A CRUD app examination for IThinkWeb Corporation
>
>Live demo: https://rivera-exam-ithinkweb.herokuapp.com/

##Node version: 14.x

## Build Setup

```cmd
# install project
$ git clone 

# open app directory
$ cd rivera-exam-ithinkweb

# install dependencies
$ npm i || npm install

# serve with hot reload at localhost
$ npm run dev

# build for production
$ npm build
```
